# DNF增幅预测器

基于贝叶斯概率更新的DNF手游增幅时机预测工具。

## 功能特点

- 🎯 **精准预测**：使用贝叶斯概率分析徽章合成结果
- 📊 **实时反馈**：即时显示好运概率和增幅建议
- 💾 **数据持久化**：自动保存历史记录到本地
- 📱 **响应式设计**：支持手机和电脑访问
- 🌐 **外网访问**：部署到Vercel提供全球访问

## 快速开始

### 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000

### 构建部署

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 使用方法

1. **选择增幅目标**：选择你要增幅的等级（如+6→+7）
2. **垫徽章**：按照提示合成对应等级的徽章
3. **记录结果**：每次徽章合成后，点击"成功"或"失败"按钮
4. **查看建议**：系统实时分析好运概率并给出建议
5. **增幅**：当建议"可以增幅了"时，进行实际增幅操作
6. **记录结果**：增幅后记录实际结果，帮助改进算法

## 算法原理

### 贝叶斯概率更新

假设游戏存在短期"运气状态"，通过观察徽章合成结果动态更新好运概率：

```
P(好运|观察数据) = P(观察数据|好运) × P(好运) / P(观察数据)
```

### 运气衰减机制

每垫3次徽章，好运概率衰减5%（最小40%），防止过度依赖历史数据。

### 预测决策

- 好运概率 > 75%：强烈建议增幅
- 好运概率 > 65%：可以增幅
- 好运概率 > 55%：可以考虑
- 好运概率 ≤ 55%：继续垫手
